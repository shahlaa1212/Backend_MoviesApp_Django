# 🎬 Movie App 
This project for move app specific backend in django

## 🛠 Installation 

### 💡 Creaate venv 
```bash
python -m venv venv
```

### 🎇 Activate venv 
```bash
source venv/bin/activate
```
### 🔧 Install requirements
```bash
pip install -r requirements.txt
```

### 📌 Make migrations 
```bash
python manage.py makemigrations
```

### ✅ Apply migrations
```bash
python manage.py migrate
```
